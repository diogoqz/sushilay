# Sushilay - Restaurante Japonés

Site do restaurante Sushilay, especializado em sushi e comida japonesa em Santa Cruz de la Sierra.

## Tecnologias

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Como executar

```sh
# Instalar dependências
npm install

# Executar em modo desenvolvimento
npm run dev

# Build para produção
npm run build
```
